# Changelog

## [1.0.2] - 2026-03-01

### 新增功能

- 增加回到顶部按钮，滚动页面后自动显示在标题栏，点击平滑滚动回顶部

## [1.0.1] - 2026-02-28

### 新增功能

- 托盘后台运行，点击关闭最小化到托盘，右键退出
- 列表新增刷新按钮，一键获取最新原型
- 新增原型时支持网络检测，连接失败时提示

### 优化改进

- 移除顶部菜单栏（File/Edit/View等）
- 卡片操作简化：移除「修改信息」按钮，改为「在线查看」
- 新增原型表单优化：公开访问默认勾选，密码框自动隐藏
- 复选框样式修复，现在正确显示且文字垂直居中

### Bug 修复

- 修复 URL 双斜杠问题（如 `http://host:8000//projects/`）
- 修复模态框输入被遮挡问题
- 修复新增原型时按钮加载状态报错

## [1.0.0] - 2026-02-28

### 初始版本发布

- 登录认证与服务器配置
- 项目列表展示与搜索
- 本地目录关联 Axure 原型
- 一键打包上传/更新原型
