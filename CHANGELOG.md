# Changelog

## [1.0.4] - 2026-03-03

### Bug 修复

- 修复项目标签配色问题，参考 AxHost 主项目实现
  - 添加预定义的标签颜色映射表 (`TAG_COLORS`)
  - 使用基础色作为背景，对应高亮色作为文字颜色
  - 添加淡色边框提升视觉效果

### 新增功能

- 项目列表支持分页加载
  - 每页加载 12 条数据
  - 滚动到距离底部 100px 时自动加载下一页
  - 加载更多时显示底部加载指示器
  - 搜索时自动重置分页并重新加载
- 点击 Header 中的「AxHost」标题可在默认浏览器打开网页端

## [1.0.3] - 2026-03-02

### Bug 修复

- 修复 macOS 图标不显示的问题，使用 `icon-mac.png` 作为 Dock/Tray 图标
- 修复 macOS 无法退出应用的问题（关闭到托盘功能改为 Windows 专属行为）

### 平台兼容性改进

- 添加平台检测逻辑，根据系统自动选择正确的图标格式
  - Windows: `.ico`
  - macOS: `.png`
  - Linux: `.png` 或 `.ico` 回退
- 添加 `before-quit` 事件处理器，确保应用可以正常退出
- macOS 点击 Dock 图标可正确恢复窗口

## [1.0.2] - 2026-03-01

### 新增功能

- 增加回到顶部按钮，滚动页面后自动显示在标题栏，点击平滑滚动回顶部

## [1.0.1] - 2026-02-28

### 新增功能

- 托盘后台运行，点击关闭最小化到托盘，右键退出
- 列表新增刷新按钮，一键获取最新原型
- 新增原型时支持网络检测，连接失败时提示

### 优化改进

- 移除顶部菜单栏（File/Edit/View等）
- 卡片操作简化：移除「修改信息」按钮，改为「在线查看」
- 新增原型表单优化：公开访问默认勾选，密码框自动隐藏
- 复选框样式修复，现在正确显示且文字垂直居中

### Bug 修复

- 修复 URL 双斜杠问题（如 `http://host:8000//projects/`）
- 修复模态框输入被遮挡问题
- 修复新增原型时按钮加载状态报错

## [1.0.0] - 2026-02-28

### 初始版本发布

- 登录认证与服务器配置
- 项目列表展示与搜索
- 本地目录关联 Axure 原型
- 一键打包上传/更新原型
